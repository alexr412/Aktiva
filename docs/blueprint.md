# **App Name**: Aktvia

## Core Features:

- Location Detection: Automatically detect the user's current geographical location using browser capabilities to enable localized searches.
- Nearby Places Search: Utilize the Geoapify API (with the provided key 'a2647722239d48c78a6ac13dfc694656') to find and display places of interest close to the user's detected location.
- Category Filtering: Allow users to filter nearby places by categories such as restaurants, cafes, parks, shops, etc., enhancing the search experience.
- Interactive Map View: Display discovered places on an interactive map, enabling users to visually explore their surroundings.
- Place Details Display: Show essential details for each place, such as name, address, and rating, upon selection from the map or list.
- Personalized Place Recommendations (AI Tool): Provide a tool to generate brief, context-aware descriptions or recommendations for a selected nearby place based on its characteristics and user preferences (if available).

## Style Guidelines:

- Light color scheme with a fresh, modern green as the primary hue. Primary color: `#39B889` (a vibrant yet calm, slightly teal green) for an active and clean feel.
- Background color: `#EEF6F3` (a very light, almost off-white with a hint of green) for a spacious and readable canvas.
- Accent color: `#17C2D9` (a bright, modern cyan/teal) to highlight interactive elements and provide a refreshing contrast.
- Headline and body font: 'Inter' (a grotesque sans-serif) for its modern, clean, and highly readable qualities across all text.
- Use clear, minimalistic, and modern solid or outline icons for navigation, categories, and map markers, maintaining a cohesive visual language.
- Employ a clean, card-based layout for displaying place details, featuring ample whitespace for readability and a responsive design that adapts seamlessly to different screen sizes.
- Incorporate subtle and smooth animations for map interactions (e.g., zooming, panning) and transitions between lists and detail views to enhance user engagement.
